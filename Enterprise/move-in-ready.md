# <a name="move-in-ready"></a>中移动已准备好

 **摘要：**获得详细信息，在云套上的内置微软云产品的存储选项。
  
"移入已准备好"的存储解决方案：
  
- 会立即使用预组包的解决方案
    
- 都准备就绪，与现有的服务捆绑在一起
    
- 提供有限的自定义项
    
以下各节描述的每个"准备搬入"存储详细信息选项。
  
## <a name="sharepoint-online-in-office-365"></a>Office 365 中的 SharePoint 在线

### <a name="features"></a>功能

- 文件从多个设备，从任何地方是安全、 可靠且可以访问
    
- 内容是通过搜索可很容易地发现
    
- 工作流和保留策略使其易于使用和维护内容
    
- 可通过 web Api 或浏览器，访问或脱机内容
    
### <a name="common-uses"></a>常见用途

- 企业或组织的门户
    
- 基于团队的协作
    
- 企业级搜索
    
### <a name="key-storage-scenarios"></a>密钥存储方案

- 合作
    
- 公司通讯
    
- 管理视频
